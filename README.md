# Tugas_Pemrograman_Basis_Data
Deskripsi

Studi kasus ini membahas pembuatan program sederhana untuk memeriksa kelayakan akademik mahasiswa sebelum pengisian Kartu Rencana Studi (KRS). Program dibuat menggunakan pemrograman prosedural di MySQL dengan memanfaatkan blok BEGIN...END, variabel lokal, dan struktur kontrol.

Tujuan
Memahami konsep pemrograman prosedural di MySQL
Menggunakan variabel lokal dan konstanta
Menerapkan logika percabangan (IF...ELSE)
Mengolah dan menampilkan data akademik mahasiswa
Melakukan analisis dan perbandingan data

Tools yang Digunakan
MySQL
XAMPP (phpMyAdmin)

Fitur Program
Menampilkan identitas mahasiswa
Validasi data akademik (status pembayaran, SKS, semester)
Kategori beban studi (Ringan, Sedang, Padat)
Kategori performa akademik (Sangat Baik, Baik, Cukup, Perlu Pembinaan)
Ringkasan kelayakan pengambilan KRS
Analisis perbandingan dua mahasiswa

Cara Menjalankan
Buka XAMPP dan aktifkan MySQL
Masuk ke phpMyAdmin
Jalankan script SQL yang telah dibuat
Gunakan perintah CALL nama_procedure();
Lihat hasil output yang ditampilkan

Contoh Output
Menampilkan identitas mahasiswa
Status kelayakan KRS (Layak / Tidak Layak)
Kategori beban studi dan performa akademik
Hasil perbandingan dua mahasiswa

Kesimpulan
Program berhasil menunjukkan bahwa pemrograman prosedural di MySQL dapat digunakan untuk mengolah data akademik secara terstruktur, melakukan validasi, serta menghasilkan informasi yang jelas dan mudah dipahami.
